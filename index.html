Introduction

Irfan's Approval System is a comprehensive approval workflow management system for Laravel applications. It provides a flexible and configurable way to implement multi-layer approval processes in your application.

The system is designed to be easily integrated into any Laravel application and provides both backend and frontend components for a complete solution.

Features

Multi-layer Approval

Support for multiple approval layers with configurable approvers and conditions.

Approval History

Comprehensive tracking of all approval actions with timestamps and remarks.

Notifications

Real-time notifications for approval requests and status updates.

Security

Role-based access control and rate limiting for secure operations.

API Integration

RESTful API endpoints for seamless integration with frontend applications.

Responsive UI

Modern, responsive user interface components built with Vue.js and Tailwind CSS.

Installation

Backend Installation

composer require irfan/approval-system

Publish the configuration and migrations:

php artisan vendor:publish --tag=approval-system-config
php artisan vendor:publish --tag=approval-system-migrations

Frontend Installation

npm install irfan-approval-system
# or
yarn add irfan-approval-system

Configuration

Backend Configuration

Configure your approval settings in config/approval-system.php:

return [
    'default_status' => 'pending',
    'actions' => [
        'approve' => 'Approve',
        'reject' => 'Reject',
        'review' => 'Review'
    ],
    'statuses' => [
        'pending' => 'Pending',
        'approved' => 'Approved',
        'rejected' => 'Rejected',
        'reviewed' => 'Reviewed'
    ],
    'rate_limit' => [
        'enabled' => true,
        'max_attempts' => 60,
        'decay_minutes' => 1
    ]
];

Frontend Configuration

Configure the frontend in your Vue application:

import { createApp } from 'vue';
import ApprovalSystem from 'irfan-approval-system';
import 'irfan-approval-system/dist/style.css';

const app = createApp(App);
app.use(ApprovalSystem, {
    apiBaseUrl: '/api',
    endpoints: {
        approvals: '/approvals',
        settings: '/approval-settings'
    }
});

Usage

Backend Usage

Add the approval trait to your model:

use Irfan\ApprovalSystem\Traits\HasApprovalWorkflow;

class YourModel extends Model
{
    use HasApprovalWorkflow;
}

Frontend Usage

Use the approval components in your Vue templates:

<template>
    <div>
        <approval-status
            :status="item.status"
            :current-layer="item.currentLayer"
            :total-layers="item.totalLayers"
        />
        <approval-actions
            :model-type="modelType"
            :record-id="recordId"
            :can-approve="canApprove"
            @approval-complete="handleApprovalComplete"
        />
    </div>
</template>

Components

ApprovalStatus

Displays the current approval status and progress.

<approval-status
    :status="status"
    :current-layer="currentLayer"
    :total-layers="totalLayers"
/>

ApprovalActions

Provides approve/reject/review actions with modal confirmations.

<approval-actions
    :model-type="modelType"
    :record-id="recordId"
    :can-approve="canApprove"
    @approval-complete="handleApprovalComplete"
/>

ApprovalHistory

Shows the complete approval history with timestamps and remarks.

<approval-history
    :model-type="modelType"
    :model-id="modelId"
/>

API Reference

GET /api/approvals/pending

Get pending approvals for the current user.

Response:
{
    "data": [
        {
            "id": 1,
            "model_type": "App\\Models\\Document",
            "model_id": 123,
            "status": "pending",
            "current_layer": 1,
            "total_layers": 3
        }
    ]
}

POST /api/approvals/approve

Approve an item.

Request:
{
    "model_type": "App\\Models\\Document",
    "model_id": 123,
    "remarks": "Approved as per policy"
}

Examples

Basic Implementation

// Model
class Document extends Model
{
    use HasApprovalWorkflow;
}

// Controller
class DocumentController extends Controller
{
    public function approve(Request $request, Document $document)
    {
        $document->approve($request->user(), $request->remarks);
        return response()->json(['message' => 'Document approved']);
    }
}

// Vue Component
<template>
    <div>
        <approval-status :status="document.status" />
        <approval-actions
            :model-type="'Document'"
            :record-id="document.id"
            :can-approve="canApprove"
        />
    </div>
</template>
